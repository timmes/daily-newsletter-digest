# Daily Newsletter Digest

An AI-powered daily digest of my newsletter subscriptions. Every morning at 06:00 CET, an agent reads all incoming newsletters, ranks the best stories, tracks trends, and publishes a curated digest.

## How it works

1. **Read** - Fetches unread newsletters from Gmail
2. **Rank** - Scores articles by topic relevance and source priority
3. **Digest** - Produces a markdown summary with top stories, trends, and quick links
4. **Deliver** - Pushes to this repo, emails the HTML version, sends a WhatsApp summary

Built with [NanoClaw](https://github.com/qwibitai/nanoclaw) and Claude.
